TANK | WILD TRAIL — мини-сайт формы регистрации v26

Что это:
- index.html — сама форма, логотип и документы согласий встроены внутрь.
- sw.js — service worker для работы без интернета после первого открытия сайта.
- manifest.webmanifest + icon-*.png — файлы PWA, чтобы сайт можно было добавить на экран «Домой».

Как использовать:
1. Загрузите ВСЕ файлы из этой папки на HTTPS-хостинг.
   Подойдут: ваш корпоративный сервер, Netlify, Vercel, GitHub Pages, Timeweb/Beget и т.д.
2. Откройте ссылку вида https://ваш-домен/index.html на iPad/iPhone в Safari.
3. Один раз откройте сайт при наличии интернета.
4. В Safari нажмите «Поделиться» → «На экран Домой».
5. После этого сайт будет открываться с экрана «Домой» и работать без интернета на этом устройстве.

Важно:
- Если устройство ни разу не открыло сайт онлайн, оно не сможет открыть его без интернета.
- Данные сохраняются локально на конкретном устройстве/браузере. Если регистрация ведется на нескольких планшетах, экспорт нужно делать с каждого планшета отдельно.
- Если открыть index.html как локальный файл из приложения «Файлы» или из мессенджера, iOS может заблокировать JavaScript. Для полноценной работы нужна HTTPS-ссылка.
